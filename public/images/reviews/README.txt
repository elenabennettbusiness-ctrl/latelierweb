Reviews section background image
================================

Drop your lifestyle photo here as:  bg.jpg  (or  bg.webp)

Recommended:
  - happy people dining in an elegant restaurant atmosphere
  - warm lighting, luxury feel
  - slightly soft / blurred (heavy blur is also OK — the overlays below
    will deepen the mood further)
  - 1920x1280 or larger, JPEG 75-85%

To use a different format or filename, update the URL in
src/components/restaurant/Reviews/ReviewsSection.jsx
(the `backgroundImage` property on the `.reviews-bg` div).
The component itself does not need any code changes — just replace
this file.
